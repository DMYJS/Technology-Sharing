简介

N_m3u8DL-CLI使用说明书

2019年9月9日更新 | nilaoda

N_m3u8DL-CLI是一款用于下载m3u8的命令行工具，基于.NET Framework 4.6框架开发，调用系统类库进行HTTP下载，支持常见的m3u8格式下载，支持从视频中选择一部分下载，还提供丰富的命令行选项以增强其功能。
正在不断更新中……

系统支持

目前仅支持WindowsXP及以上系统(若您坚持WindowsXP，将有部分资源无法下载。我强烈推荐您使用Windows7及以上版本)

程序依赖

所需运行时环境：.NET Framework 4.6
用于合并分片：ffmpeg 或我编译的精简版

https://github.com/nilaoda/N_m3u8DL-CLI/releases
